## 🎮 产品体验流程

### Step 1：创建你的数字人格
填写/生成用户画像：
- 职业、性格（MBTI）、星座、当前心情
- **时薪设定**：定义你的时间价值（影响Token换算）
- Token分配策略：工作、娱乐、健康各分配多少资源

### Step 2：生成一小时AI任务清单
系统根据你的画像，自动生成8-10个OPC任务，例如：
- 📊 工作AI：根据财务数据生成月度报表
- 🎬 娱乐AI：生成一个3分钟的调酒教程视频
- 💤 健康AI：设置智能睡眠监测计划
- 💰 财富AI：比价全网机票，找出最优方案
- 📖 意义AI：给久未联系的朋友写一封信

**特别设定**：
- 有高耗Token的任务（Level A需要200 Token）
- 有Deadline任务（明天出差，今天必须完成攻略）
- 有"私房钱"储备金（应急用）

### Step 3：开启议会，观战AI博弈
点击"开始议会"按钮，进入实时对话流：
```
🎯 ENTJ（效率之神）: 
"我要把'生成财报'升级到Level A，需要200 Token。
ESTP，你把'刷短视频总结'降到Level C，省出100 Token给我，
我分你50 Token做应急储备。"

🎲 ESTP（娱乐之神）: 
"凭什么？我要做调酒视频Level A，需要300 Token！
你的财报降到Level B，省50给我。成交吗？"

🛡️ ISFJ（健康守护者）: 
"你们都疯了！压缩睡眠时间？除非ENTJ答应
把'代码审查'从A降到B，省的Token买健康食谱。"

💫 INFJ（意义使者）: 
"ENTJ，全是工作任务，活着还有意义吗？
我要否决'加班做表格'，除非加一个给朋友写信的任务！"
```

**互动功能**：
- 投喂Token：给某个AI注入资源，改变谈判力量
- 上帝否决：直接驳回某个提案
- 每2分钟自动播报：当前博弈进度 + 资源分配情况

### Step 4：收获终极优化报告
一小时倒计时结束后，收到：
1. **任务执行报告**：每个任务的最终等级 + Token消耗明细
2. **事件结算卡片**：每个任务的随机结算结果
   - 🎉 大成功：演讲稿意外爆红，社会影响力+10
   - ✅ 成功：任务完成，符合预期
   - ⚠️ 勉强通过：逻辑不通，职业声望-2
   - ❌ 大失败：AI死机，当众出丑
3. **博弈回放**：完整的AI谈判过程存档

## 附录：核心机制速查表

| 概念 | 定义 | 示例 |
|------|------|------|
| **Token** | 算力货币，用于交易任务质量 | 1 Token = 0.05 RMB |
| **Level C** | 低质量完成（敷衍） | 复制粘贴携程攻略 |
| **Level B** | 标准完成（合格） | 根据偏好筛选 + 简单排期 |
| **Level A** | 高质量完成（完美） | 实时爬取 + 3D预览 + 自动文案 |
| **Deadline任务** | 必须完成的关键任务 | 明天出差，今天必须生成攻略 |
| **私房钱** | AI的应急储备Token | 用于紧急交易或兜底 |
| **讨价还价** | AI必须提出具体交易方案 | "我给你X，你给我Y" |
| **事件结算** | 任务完成后的随机评级 | 大成功/成功/勉强/大失败 |
